Phụ lục:
Yêu cầu tạo 1 file exercise1.php để thực hiện yêu cầu trên. File trên sẽ được add vào repo đã được tạo ở bài tập về git, với:
tên branch: feature-tên_bạn-php-basic (VD: feature-sonnt-php-basic)
message commit: PHP basic
Tuân thủ coding convention như sau https://docs.google.com/document/d/14VfbgEO_yH5TfEK23PQxFMbjluqF4Wdt0BeaAbKpaYI/edit
file1.txt, file2.txt là 2 file đã được thêm với nội dung tùy ý ở bài tập về git. Ở bài này cần được sửa nội dung như đã mô tả ở bài 2. Ở đây cần đọc file file1.txt và file2.txt để làm đầu vào cho function đã viết.
Tạo merge request từ branch trên vào branch master. Gửi link merge request cho người chấm bài.
